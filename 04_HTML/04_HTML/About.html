Problem Definiton

We want to develop a machine learning model which will identify from the news source which is or are producing fake news. We will build a classifier which will detect or make descisions about the information that were given in the news articles, we will be using multiple resources.


Hypothesis

The model will be able to predict whether an article contains fake news based on the multiple sources

Null Hypothesis

The model was not able to predict whether an article contains fake news based on the multiple sources.
