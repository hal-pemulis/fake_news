Overview

This project and app will use machine learning to analyze fake news articles to discover meaningful insights and possibly detect if text is "fake news." We will use labeled datasets to train a machine learning model. This project will use natural language processing to detect fake news directly, based on the text content in news articles from a variety of sources.


Problem definition

There are many categories that misinformation falls into, including articles that are completely false, articles that provide truthful information but make false interpretations, and articles that provide opinions disguised as news. There are also articles from tweets and quotes from people other than news organizations. For this project, we define "fake news" as containing inflammatory or sensationalized information. This should be detectable with an ML model trained using tf–idf features/words.


Hypothesis

The model was able to predict whether an article contains fake news based on tf–idf input.

Null Hypothesis

The model was not able to predict whether an article contains fake news based on tf–idf input.
